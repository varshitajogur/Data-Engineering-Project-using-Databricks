# **FMCG Data Engineering Pipeline Using Databricks**

An **End-to-End Data Engineering** project that demonstrates a real-world ETL pipeline in the **FMCG** domain. The project simulates a retail company acquisition by integrating data from two organizations into a unified **Lakehouse Architecture** using the **Medallion Architecture (Bronze, Silver, Gold)**.

---

## **Project Overview**

This project focuses on building a scalable data pipeline that ingests, transforms, and consolidates data from multiple sources into a centralized Lakehouse. The pipeline follows modern data engineering practices to enable reliable analytics and reporting.

---

## **Architecture**

* **Bronze Layer** – Raw data ingestion
* **Silver Layer** – Data cleansing and transformation
* **Gold Layer** – Business-ready datasets for analytics

---

## **Features**

* End-to-End ETL Pipeline
* Lakehouse Architecture
* Medallion Data Model
* Data Cleansing and Transformation
* Scalable Data Processing with Apache Spark
* Interactive BI Dashboard
* Genie Integration for Data Insights

## **Project Workflow**

1. Ingest raw data from multiple sources.
2. Store raw data in the **Bronze Layer**.
3. Clean and transform data in the **Silver Layer**.
4. Create analytics-ready datasets in the **Gold Layer**.
5. Visualize business insights using **Power BI**.
6. Query and analyze data using **Genie**.
